# Discipline

Discipline is an Expo React Native calorie tracking app backed by Supabase.

## Features

- Email/password authentication with Supabase Auth
- User profiles with daily calorie goals and color-coded progress
- Camera/gallery meal photo capture
- Private Supabase Storage uploads for meal photos
- NVIDIA NIM-powered meal analysis through Supabase Edge Functions
- Meal feed, meal detail, and settings

## Local Setup

Install dependencies:

```bash
npm install --legacy-peer-deps
```

Create `.env` from `.env.example`:

```bash
cp .env.example .env
```

Fill in:

```bash
EXPO_PUBLIC_SUPABASE_URL=
EXPO_PUBLIC_SUPABASE_ANON_KEY=
```

Do not commit `.env`.

## Supabase

Link your Supabase project:

```bash
supabase link --project-ref YOUR_PROJECT_REF
```

Push migrations:

```bash
supabase db push
```

Required Edge Function secret:

```bash
supabase secrets set NVIDIA_API_KEY=YOUR_NVIDIA_API_KEY
```

Optional model override:

```bash
supabase secrets set NVIDIA_MODEL=meta/llama-4-maverick-17b-128e-instruct
```

Deploy functions:

```bash
supabase functions deploy analyze-meal
supabase functions deploy get-upload-url
supabase functions deploy get-photo-url
supabase functions deploy get-photo-urls
```

## Development

Run all update/deploy steps and start Expo:

```bash
npm run update:launch
```

Or just start Expo:

```bash
npm start
```

Run checks before pushing:

```bash
npm run typecheck
npm run security:check
```

## Notes

- Meal photos are stored in the private `meal-photos` Supabase Storage bucket.
- `meals.photo_url` stores a private object key, not a public URL.
- Signed URLs are generated by Edge Functions when photos need to be displayed or analyzed.
